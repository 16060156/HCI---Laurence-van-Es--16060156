Processing

De volgende opdrachten heb ik uitgevoerd in Processing. Dit is een programma waarmee je figuren kunt maken door middel van code schrijven. Ook interactieve animaties kunnen met Processing worden gemaakt. Processing hanteert de programmeertaal Java.

Opdracht 1
Write a program that draws three (or more) ellipses:
• At various locations,
• With various colors;
• Various widths and heights;
• With and without borders (of various colors).

Opdracht 2
Write a program that draws:
• At least four different shapes;
• In various colors;
• At various locations.

Opdracht 3
Write a program that draws at least two different shapes, with various colors and make them interactive. Ideas:
• Draw a line where one of the ends follows the mouse;
• Draw a circle where the color is related to the location of the mouse;
• Draw a rectangle with the size related to the x location of the mouse;
• Etc.
