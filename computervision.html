Computer Vision

Op deze pagina kunt u de Processing & Computer Vision opdrachten vinden.

Processing

De volgende opdrachten heb ik uitgevoerd in Processing. Dit is een programma waarmee je figuren kunt maken door middel van code schrijven. Ook interactieve animaties kunnen met Processing worden gemaakt. Processing hanteert de programmeertaal Java.

Opdracht 1
Write a program that draws three (or more) ellipses:
• At various locations,
• With various colors;
• Various widths and heights;
• With and without borders (of various colors).


Opdracht 2
Write a program that draws:
• At least four different shapes;
• In various colors;
• At various locations.


Opdracht 3
Write a program that draws at least two different shapes, with various colors and make them interactive. Ideas:
• Draw a line where one of the ends follows the mouse;
• Draw a circle where the color is related to the location of the mouse;
• Draw a rectangle with the size related to the x location of the mouse;
• Etc.

Computer Vision

Voor deze opdrachten werk ik nog steeds in Processing. Er zijn bestaande libraries die gedownload kunnen worden, bijvoorbeeld libraries met code die iets via de webcam kunnen waarnemen en tracen.

Opdracht CV1
Now it's time to start experimenting. Take a good look at the LiveCamTest code you just opened.

Can you discover which line causes a rectangle to be drawn around a recognized face? (Tip: rect = rectangle).
Adjust the code so that a circle is drawn instead of a rectangle.


Opdracht CV2
Adjust the code so that every recognised face gets an 'anonymous' black bar in front of his eyes or a red clown nose instead of his/her nose.


Opdracht CV3
Adjust the code of one of the examples so that something cool happens.


CV bonus
Adjust the code so that every recognised image is enhanced by an image e.g. give a recognized face funny glasses or a hat.
